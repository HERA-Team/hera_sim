============
Installation
============

Requirements
------------
Requires:
 * `numpy`
 * `scipy`
 * `aipy`
 * `hera_cal` (which requires `h5py`)
 * `pyuvdata`

Then, at the command line, navigate to the `hera_sim` repo/directory, and::

    pip install .

If developing, from the top-level directory do::

    pip install -e .
